# CONTRIBUTING TO COLLECTIVE

Collective is a meritocracy of utility. We do not "assign" work; we issue **System Calls**.

## How to Contribute

1. **Explore the Roadmap:** Visit our [Project Board](https://github.com/raffaine/collective/projects).

2. **Claim a Story:** Comment on any open Issue with `/claim`. An Architect will verify your "Skill Weight" and tag you.

3. **Submit a Branch:**
   - For **Mythos**: Submit LaTeX updates to `/doc/book_syscall`.
   - For **Logic**: Submit Schema or Whitepaper updates to `/doc/whitepaper`.
   - For **Code**: Submit Python/Rust/JS tools to the `/src` directory.

## The Code of Conduct

- **Utility over Profit:** No code shall be accepted that prioritizes capital accumulation over human metabolism.

- **Transparency:** All logic must be auditable by any Peer.

- **Resilience:** Design for the "Maintenance Lockout." If it requires the cloud to function, it is not Sovereign.
